# NordOrd Mix 1700

新版瑞典语混合词库网页。

## 特点

- 去掉 A1/A2/B1 难度入口
- 约 1700 个混合词条：单词、短语块、句型
- 每轮可选择 20 / 50 / 100 / 200 / 全部
- 支持主题筛选
- 支持搜索
- 支持瑞典语发音
- 支持错题本
- 使用 localStorage 保存进度
- 适合 GitHub Pages 部署

## 部署到 GitHub Pages

将这些文件上传到原仓库根目录，覆盖旧文件：

- index.html
- style.css
- app.js
- words.json
- README.md

等待 GitHub Pages 自动更新即可。

## 注意

大规模词库为结构化生成，适合自用背诵。如果要用于考试、出版或正式教学，建议逐批校对。
